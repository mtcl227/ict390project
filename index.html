Welcome

Welcome to our website about Drupal, which is ironically not made with Drupal. Here we'll introduce just what Drupal is as well as a tutorial on one of countless Drupal modules.

But what is Drupal? Well, Drupal is an open source framework for making websites. Instead on mucking about with HTML code, you can muck about with various modules to customize your own site!

Okay, okay, but really: Drupal is a powerful tool that requires a different skillset to use than coding an HTML page. It is more like using Legos to build a site, placing all the pieces just so, to create a respectable final product. For the truly committed, you can use programming knowledge and the open source code of Drupal to write new modules. Writing modules is a bit out of scope for us here, however. But let's get into an actual tutorial. If you'd be so kind as to head on over to our Tutorials page, we'll get started on installing a module.
